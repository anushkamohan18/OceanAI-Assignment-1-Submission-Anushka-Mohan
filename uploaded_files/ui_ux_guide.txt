# UI/UX Guidelines

## Colors
- Primary Button (Pay Now): Green (#28a745). Hover: #218838.
- Error Text: Red.
- Success Message: Green.

## Form Validation
- All fields (Name, Email, Address) are required.
- Email must contain '@'.
- Error messages should appear below the invalid field.
- "Pay Now" button should disappear on successful submission.

## Layout
- Two-column layout on desktop: Form on left, Cart on right.
- Responsive design for mobile.
